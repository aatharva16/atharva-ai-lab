## How I use this repo

- **Daily commit rule:** Every commit must represent something real — a prompt, a script, a note on what I learned, a tool I built. No empty commits.
- **Weekly LinkedIn post:** Every Sunday I post one thing I shipped that week, linking back here.
- **Monthly review:** I look back at the month and assess if I'm actually levelling up.
