# stock-tools 📈

AI-powered tools for Indian equity research and investing.

## What lives here

- Sovrenn valuation calculators (Large Order, Uptrend, Capacity Expansion, Preferential Allotment)
- Aditya Joshi 6-criteria stock screener outputs
- Prompt templates for financial analysis
- Stock analysis notes and dashboards

## Context

I invest in Indian small/micro-cap equities across Zerodha, Groww, IndMoney, and Dhan.  
Building tools to speed up my research workflow using the Sovrenn methodology.
