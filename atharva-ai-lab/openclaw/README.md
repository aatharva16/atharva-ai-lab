# openclaw 🤖

My personal AI OS — six specialized agents running on Telegram via OpenRouter.

## What lives here

- Agent configs and system prompts
- Architecture notes and decisions
- Cron job setups
- Tool integrations and learnings

## Context

Deployed on Hetzner CAX11 (ARM64). Six agents: Health Tracker, Stock Analyser,  
Portfolio Checker, News Feed, Job Searcher, Email Organiser.  
Using OpenRouter as unified LLM provider with Telegram as the interface.
